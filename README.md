# AI-BASED-TIME-TABLE-GENERATION-SYSTEM-WITH-NEP2020-FOR-MULTIDISCIPLINARY-EDUCATION-STRUCTURE
Project Overview
This project is a web application that uses Artificial Intelligence to automatically generate a conflict-free college timetable. It follows the NEP 2020 guidelines by supporting multidisciplinary courses, open electives, flexible credit systems, faculty preferences, and efficient classroom allocation. AI helps reduce manual work while creating optimized schedules. AI-based scheduling methods are well suited to handling the additional flexibility introduced by NEP 2020. �
